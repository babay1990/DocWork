Добро пожаловать в ознакомительную версию приложения DocWork!


Приложение представляет из себя простую версию документооборота, предоставляющую возможность обмениваться сообщениями, создавать поручения, служебные записки и т.п. При разработке использовались следующие технологии:

- Java 8;

- Spring Boot;

- Spring Security;

- PostgreSQL;

- Thymeleaf;

- AmazonAWS API;

- JSoup.
